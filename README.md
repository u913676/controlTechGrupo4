# Controltech

📌 Proyecto: [Controltech en Google Stitch](https://stitch.withgoogle.com/projects/6889338777581620843)

## Descripción
Controltech es una aplicación desarrollada para la gestión de activos tecnológicos dentro de una organización.  
Permite registrar, listar y consultar el detalle de cada equipo, facilitando el control patrimonial y el seguimiento de su estado.

## Funcionalidades principales
- ✅ Registrar un equipo en el inventario global.  
- 📋 Listar todos los equipos registrados.  
- 🔎 Consultar el detalle de cada equipo (información técnica, estado, ubicación, usuario asignado, etc.).
